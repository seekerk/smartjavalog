# smartjavalog

## Usage
usage: smartjavalog
 -i,--input <arg>      input owl file path
 -n,--name <arg>       package name
 -o,--output <arg>     output folder path
 -p,--platform <arg>   target platform. Supports "android" or "default"
                       values
